# Face recognition authentication 

---

## Backend Flask

### Init

```
pip install flask
pip install flask_cors
```  

### Run 

```
cd backend
flask run
```

> Running on http://127.0.0.1:5000

---

## Frontend

### Init

```
npm install -g @vue/cli

cd frontend
npm install
```

### Run  

```
npm run dev
```
>Local:   http://127.0.0.1:3000/
